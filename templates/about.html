MealMatch

About MealMatch

MealMatch is a smart web application designed to assist users in discovering meals they can prepare based on the ingredients they already have at home. It aims to minimize food wastage, simplify meal planning, and promote efficient grocery utilization. The core functionality includes an AI-powered ingredient detection system. Users can upload an image of an ingredient, and our trained convolutional neural network (CNN) model classifies the ingredient to help in generating suitable recipes based on it. This feature is especially useful for quick meal decisions without needing to manually enter ingredient names. Users can create accounts to access a personalized dashboard, upload ingredient images, view detected items, and explore potential recipes. The project integrates both frontend and backend seamlessly, using Flask, HTML/CSS, and machine learning models.

MealMatch is a step toward smarter, AI-assisted kitchens that prioritize sustainability, convenience, and personalization.
